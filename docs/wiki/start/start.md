## 第一步：下载 智绘教Inkeys

智绘教Inkeys 支持 Windows 7 RTM(sp0) 及以上版本。

### 下载

1. 选择合适的版本并选择下载  
  
<DownloadCard />

2. 下载完成后会得到一个压缩包。  
  
![](/wiki/start/3.png){ width="50%" }   

### 解压

1. 使用解压软件，将软件解压出来。  
  
![](/wiki/start/4.png){ width="50%" }  


2. 打开解压后的文件夹，即可看到运行文件。  
  
![](/wiki/start/5.png){ width="50%" }

## 启动 智绘教Inkeys

双击 `Inkeys.exe` 即可启动程序。  

![](/wiki/start/7.png)

您将看到软件的悬浮窗（在屏幕右下角）。点击后可以展开主栏，那么恭喜您软件安装成功了！
